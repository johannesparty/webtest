Overview

The export API returns site data in JSON format with the following structure. Click on field names to jump to their documentation.

sites[]
  project
    soilSettings
      depthIntervals[]
  soilData
    _depthIntervals[]
  notes[]
    author
  soil_id
    soilMatches
      matches[]
        combinedMatch
        dataMatch
        locationMatch
        soilInfo
          soilSeries
          ecologicalSite
          landCapabilityClass
          soilData
            depthDependentData[]

Fields starting with _ are derived fields expanded by the export for convenience.
