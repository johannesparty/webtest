LandPKS CSV Export Format

Documentation for the CSV format returned by the export API

Site Project Location Soil Matches Ecological Site Soil Data Depth Intervals See also: JSON Format

Overview

The CSV export produces one row per depth interval per site. Site-level fields repeat on each row.

Site

CSV ColumnTypeDescription
Site IDstringUnique identifier for the LandPKS Soil ID site
Site namestringName of the LandPKS Soil ID site
Site privacyPrivate, PublicPrivacy setting of the site or its parent project

Project

CSV ColumnTypeDescription
Project IDstringUnique identifier for the LandPKS Soil ID project
Project namestringName of the LandPKS Soil ID project
Project descriptionstringDescription of the LandPKS Soil ID project

Location

CSV ColumnTypeDescription
LatitudenumberLatitude of the LandPKS site (-90 to 90)
LongitudenumberLongitude of the LandPKS site (-180 to 180)
ElevationnumberHeight of the site above mean sea level (meters)

Soil Matches

CSV ColumnTypeDescription
Selected soil seriesstringSoil series selected by the user as best match (e.g., "Literr", "Togasara")
Selected soil type taxonomy subgroupstringSubgroup of the soil series selected by the user (e.g., "Typic Argixerolls")
Selected soil descriptionstringDescription of the soil selected by the user
Top soil series matchstringSoil series suggested by the algorithm as best match
Top soil match taxonomy subgroupstringSubgroup suggested by the LandPKS Soil ID algorithm (e.g., "Pachic Argixerolls")
Top soil match descriptionstringDescription of the soil suggested by the algorithm
Top match user ratingstringUser's feedback on the top algorithmic match: Selected, Rejected, or Unsure

Ecological Site

CSV ColumnTypeDescription
Ecological site namestringEcological site name (e.g., "Loamy Upland", "Sandy Bottomland")
Ecological site IDstringEcological site ID (e.g., "R042XB012NM")
Land capability classificationstringClass 1-8 (1=fewest limitations, 8=most severe) + subclass: e=erosion hazard, w=wetness/excess water, s=soil limitations (shallow, droughty, stony), c=climate. Examples: "3e", "6w"

Soil Data

CSV ColumnTypeDescription
Slope steepness percent lownumberLower estimate of slope steepness as percent rise (e.g., 5 means 5% grade or 5 ft rise per 100 ft horizontal)
Slope steepness percent highnumberUpper estimate of slope steepness as percent rise (e.g., 15 means 15% grade)
Down slopeConcave, Convex, LinearShape of the downhill slope
Cross slopeConcave, Convex, LinearShape of the slope perpendicular to downhill direction
Surface cracksNo cracking, Surface cracking only, Deep vertical cracksPresence and intensity of surface cracks
Site notesstringNotes entered for the site. Format: content | author_email | timestamp; (next note). Newlines shown as â†µ. Example: "Soil was moist after rain | jane@example.com | 2025-01-15T14:30:00+00:00; Rocky area near creek | john@example.com | 2025-01-14T10:00:00+00:00"

Depth Intervals

One row per depth interval. Intervals are determined by the site's depth interval preset.

CSV ColumnTypeDescription
Depth labelstringLabel of the soil pit depth (e.g., "0-1 cm", "1-10 cm", "10-20 cm")
Depth startnumberUpper bound of the soil pit depth interval (cm)
Depth endnumberLower bound of the soil pit depth interval (cm)
Depth sourceNRCS, BLM, CUSTOMSource preset for this depth interval
Depth rock fragment volume0-1%, 1-15%, 15-35%, 35-60%, >60%Percent of rock fragments in the soil sample
Depth texture classClay, Clay Loam, Loam, Loamy Sand, Sand, Sandy Clay, Sandy Clay Loam, Sandy Loam, Silt, Silt Loam, Silty Clay, Silty Clay LoamTexture class of the soil sample
Depth soil colorstringMunsell color values of the soil sample (e.g., "10YR 4/3", "5YR 3/2"). Computed from colorHue, colorValue, colorChroma.
Soil color photo usedTRUE, FALSEWhether soil color was determined through photo analysis
Soil color conditionMoist, DrySoil dampness (moist/dry) of the sample
Soil color lightingEven, UnevenLighting conditions of the sample photo
